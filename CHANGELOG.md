# Changelog

Wszystkie istotne zmiany w projekcie będą dokumentowane w tym pliku.

Format oparty na [Keep a Changelog](https://keepachangelog.com/pl/1.0.0/),
i projekt stosuje [Semantic Versioning](https://semver.org/lang/pl/).

## [2.3.0] - 2025-02-10

### Dodane
- Pełna refaktoryzacja struktury HTML5 z semantycznymi tagami
- Osobne pliki CSS i JavaScript (zamiast inline)
- Meta tagi SEO (description, keywords, Open Graph, Twitter Card)
- Schema.org structured data dla lepszego SEO
- ARIA labels dla dostępności (WCAG 2.1 Level AA)
- Skip to content link dla użytkowników screen readerów
- Responsywne menu mobilne z hamburger icon
- Smooth scrolling dla linków nawigacyjnych
- Active state dla aktualnej sekcji w nawigacji
- Fade-in animacje przy scrollowaniu
- Lazy loading dla obrazów
- robots.txt i sitemap.xml
- .nojekyll dla GitHub Pages
- README.md z pełną dokumentacją
- .gitignore dla repozytorium Git
- CHANGELOG.md

### Zmienione
- Poprawiono strukturę nagłówków (h1-h6)
- Zaktualizowano przyciski CTA w sekcji Hero
- Ulepszone style dla lepszej czytelności
- Zoptymalizowano responsive breakpoints
- Poprawiono kontrast kolorów dla dostępności
- Zaktualizowano footer z lepszą strukturą

### Poprawione
- Błędy w strukturze HTML
- Problemy z dostępnością
- Brakujące alt texts dla obrazów
- Problemy z wydajnością
- Cross-browser compatibility

### Usunięte
- Inline CSS (przeniesione do styles.css)
- Inline JavaScript (przeniesione do script.js)
